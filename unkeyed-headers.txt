# Unkeyed headers - common HTTP headers that AFFECT the response but
# are NOT part of typical web-cache keys. Send any of these with a
# unique value, observe whether the response changes; if the cache
# then serves your modified response to other clients, you have
# WEB CACHE POISONING.
#
# Use as the wordlist for intruder.py sniper-mode against a request
# template that has §HEADER_NAME§ in front of `: probe-value`. See
# examples/cache-poison-template.txt for the canonical template.
#
# References: PortSwigger's web cache poisoning research
# (https://portswigger.net/research/practical-web-cache-poisoning).

X-Forwarded-Host
X-Forwarded-Server
X-Forwarded-For
X-Forwarded-Proto
X-Forwarded-Scheme
X-Forwarded-Port
X-Forwarded-Path
X-Forwarded-Prefix
X-Forwarded-Ssl

X-HTTP-Host-Override
X-Originating-URL
X-Url-Scheme
X-Proxy-URL
X-Proxy-Host

# --- RFC 7239 Forwarded (multi-value, often misparsed) ---
Forwarded
